关键词：海外上市

分类：金融服务业

介绍：海外上市是指国内股份有限公司向海外投资人发行股票，在海外公开的证券交易场所流通转让。狭义的海外上市就是国内企业向海外投资者发行股权或附有股权性质的证券，该证券在海外公开的证券交易所流通转让。广义的海外上市是指国内企业利用自己的名义向境外投资人发行证券进行融资，并且该证券在境外公开的证券交易场所流通转让。
