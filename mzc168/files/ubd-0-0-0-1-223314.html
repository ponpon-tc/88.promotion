关键词：领导者素质

分类：领导与控制管理

介绍：领导者的素质，是指在先天禀赋的生理和心理基础上，经过后天的学习和实践锻炼而形成的在领导工作中经常起作用的那些基础条件和内在要素的总和。在领导科学理论的研究中，人们一般把领导者的素质分为政治素质、思想素质、道德素质、文化素质、业务素质、身体素质和心理素质，以及领导和管理能力等。
