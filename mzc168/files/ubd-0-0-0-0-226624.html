关键词：卫生保障

分类：行业

介绍：卫生保障，又称卫生医疗保障。是指职工及供养直系亲属因疾病、负伤、生育时，由社会提供必要的医疗服务和物质保障的制度。职工的病伤、生育、养老等都会遇到医疗问题，因此，它是疾病、伤残、生育、养老社会保险中一项重要内容。医疗社会保险是健康的保障。实行医疗保险对病伤职工提供医疗服务，在经济上给予物质帮助，使其尽快恢复健康和工作，是一种社会责任，对促进社会主义物质文明和精神文明建设都有着积极作用。
