关键词：大盘指数

分类：金融服务业

介绍： 中国股票大盘指数是指：沪市的“上证综合指数”和深市的“深证成份股指数”。上证综合指数：以上海证券交易所挂牌上市的全部股票（包括A股和B股）为样本，以发行量为权数（包括流通股本和非流通股本），以加权平均法计算，以1990年12月19日为基日，基日指数定为100点的股价指数。
