关键词：政府与公共事业

分类：行业

介绍：公用事业是指负责维持公共服务基础设施的公司。公共事业处在自然垄断之下，可能是由于政府的控制之下，而如果是私营的则会被行政法规所监督。一般所说公用事业包括电力、供水、废物处理、污水处理、燃气供应、交通、通讯等。
