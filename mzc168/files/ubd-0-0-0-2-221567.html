关键词：危机公关

分类：战略管理

介绍： 很多在企业做公关工作的人都抱着一个观点：做公关就是做广告，需要大量的资金投入。事实上，广告和公关的区别是很大的：广告专注的是诉求，公关借助的是沟通；广告张扬，公关内敛。区分了广告和公关的内涵，我们就做对了一半。只要将所有的问题定位在沟通上，那么我们就有可能用最低的成本来消除危机。
