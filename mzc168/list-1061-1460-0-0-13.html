“过劳死”行业中的不“牺牲”原则

　　我是广告学专业大四的学生，现在正在进行毕业实习。混迹在广告圈的人都知道，我们总是没有个下班吃饭的准点，很多人都有像胃病、神经衰弱等这样的职业病。初涉此行的我们也是深有感触，我已经连续上班半个月，每天加班到十一点。因为客户的要求和客户的到访都是我们不能控制的。当然，这样高强度的工作对于身体的损害是不言而喻的。难道说"进了广告界=等着过劳死"这句话真成了永恒不变的真理。我想请问专家如何在这样忙碌辛苦的行业，通过时间管理安排好自己的工作生活作息，摆脱身体和工作压力的双重折磨？

关键字 : 过劳死

在如此家族企业工作？别犯傻

去年夏天我毕业了，找工作并不顺利，终于一家家族式贸易公司向我伸出了橄榄枝，我欣然接受了这份工作，职位是生产专员。尽管工作辛苦工资也不是很多，可我从未抱怨过，认为这应该是一个职场新人必经的过程。但随着时间的推移，我进入公司半年之后，我越发觉得不对劲起来。老板无缘无故将我的上班时间改为六天，且无任何加班费。半年里我包揽了公司所有杂事，上吨的原料、包材等的搬运也要我上，吝啬到民工都不肯请。半年的工作时间里充满了训斥，老板每周都要把我以前做的所有事给训斥一遍。 对此我非常委屈，身心都疲惫不已。最近一次供应商的失误，老板又一次把我骂的一无是处。但联络供应商的问题他根本不放权，我没有决定权，我哪敢私自解决。导致基本我是里外不是人。这是我第一份工作，来之不易，我不知道是去是留，我该不该忍气吞声？

为生存或为职涯发展的艰难选择

　　我是今年毕业的应届本科毕业生，最近在投简历忙着找工作。去年后半年我在一家代理记账公司工作，每天都要去其他公司取凭证报账很辛苦，需要经常加班，工资也不高。我并不是本地生源，市区的而生活成本很高，交通费房租伙食费扣除后所剩无几，生活一直很拮据。但实习单位能接触到不同的记账方法，可以学到很多东西。最近有一家学校附近的公司找到我，福利待遇优厚，并且愿意让我提前拿转正工资。老板住在附近，可以接送我们学校的同学上下班，我住在学校大大节约了生活成本。于是，我欣然接受了这份审计的工作，可干了几天我就发现，因为是贸易投资公司审计工作十分清闲，又没有师傅带，我几乎学不到东西。面对生活的压力，我不知道该如何选择，到底要找一份薪资待遇好的工作，还是能学习到东西对职业规划有利的工作？请专家老师帮我支招。

应对如此加班文化的策略

我所在的公司是家本土的世界500强，工资不低待遇不差，但就是爱加班。没有事情也要加，哪怕在那里干坐着、打游戏、听歌、淘宝都成，但必须完成每个月的加班"任务"。平时被迫加班就算了，可连国家法定的节假日也要苛扣。对此大家抱怨连天，却敢怒不敢言。我想请问专家，面对公司加班文化，我该怎么办？

与老板关系暧昧的同事,惹不起?

我们大领导和我的（女）同事是很暧昧的关系，大家私底下都称此女为女神。女神是我同一个部门的平级，特别喜欢做好人，但不想做事情。于是平时我们部门的同事不但要做自己分内的事情，还要帮女神完成任务。而且若是做得不好，她还会到领导面前给你小鞋穿，让你落个不成熟，办事不利落的名声。面对这位供着很累、想躲躲不掉、又惹不起的女神，我该怎么办？

关键字 : 领导

不分早晚加班vs早晚打卡,苦恼

我们公司不是那种朝九晚五可以准时的企业，经常要加班，有的时候甚至要到半夜以后。但是公司却有严格的考勤制度，如果早上上班不打指纹，我们的收入就会有损失。想想自己平时工作很认真，而且常常加班到深夜，我的工作成绩也是有目共睹的。公司不给加班费不说，还要以考勤来克扣我。说是公司要严格纪律，如果我按时上班，表现会更好。但是我平时经常工作到半夜，而且不是我可以控制的。现在又要早上早起，我实在感到委屈。请问专家我如何面对这样的公司管理呢？

关键字 : 考勤制度

监控员工的外行压榨型老板,炒他

我在一家台资企业任工程师一职，但我们的直接领导是个外行根本不懂我们的专业，却总是干涉我们的工作，并强迫我们无偿加班，包括周末也搭上了。大家都对此有极大的不满。台资企业的老板大部分是台湾人，我们这个老板的上级也是台湾人，他们对我们大陆人的态度基本是不理睬。同时，工作时间采用监视电脑对员工的隐私进行监控。我的工作很不开心。请问专家，我该如何？

如何巧用业余时间挣外快

由于业务时间比较固定，而且面临住房和养家的压力，我决定好好利用我的业务时间。现在，我有好几个不错的项目都想做，但是我知道业余时间不多，只有专注一个方向才能容易成功。请问专家，我该如何选择项目，该放弃那些项目呢？

遭遇铁公鸡上司，弃还是忍

　　我是一个在职场工作多年的人了，最近我们部门新任命了一个领导，但是一段时间相处以后，发现这个领导非常小气，基本从来不请部门员工吃饭，就是一只铁公鸡。自己还常常上班迟到，而且有什么好事的时候，也很少想到部门员工。和这样的领导相处，我感觉没什么劲！请问专家，我该如何和这样的领导相处？

预算压力下转型,如何获老板支持?

　　最近因为集团组织架构调整，我所管理的部门并入另外一个群组，那个群组的老板对我部门所从事的业务并不是太了解。而这些年由于市场的变化，我所管理的部门面临着一些挑战。如果我们不变化，未来的前景并不乐观。但是变化需要老板给予一定的支持和投入，而我们部门现在的预算完成情况就不是太理想，请问专家，这样的情况，我该如何从我的老板那里得到更多支持？

关键字 : 预算
