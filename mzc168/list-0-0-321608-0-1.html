IPTV经营原则确定 电信联通划地而治

【文章摘要】三网融合IPTV业务经营原则确定，原则上，在某一个试点地区，只能存在一个广电运营商和一个电信运营商进行竞争合作。而对于参与三网融合的电信运营商而言，中国电信和中国联通将划地而治。

电信运营商延迟出台2010年投资计划

在日前举办的上海贝尔上下游产业对接研讨会上，上海贝尔高级执行副总裁徐智群向记者透露，直到现在国内三大电信运营商还未拿出2010年具体投资计划，这在以往是非常罕见的。他表示，未来三年，国内运营商对待网络扩容的态度将更加审慎，而运营商投资减少，对于设备商而言影响巨大。

关键字 : 电信运营商电信商

三大电信运营商转战“软件商店”

随着3G时代的到来，中国三大电信运营商先是在上网本方面展开一番火拼，如今又将战火延至“软件商店”。继中移动日前确定将于9月正式上线Mobile Market后，记者还获悉，中国电信也悄然涉足“软件商店”领域。

关键字 : 电信运营商

工信部证实将于电信重组结束后发放3G牌照

7月17日下午4时，与“2008北京奥运会”奥组委举行的奥运通信服务与保障新闻发布会上，工业和信息化部副部长奚国华明确表示，业界关注已久的3G牌照将于电信重组后发放。” 　　由于目前尚未发放3G标准，WCDMA和CDMA2000两大3G标准的手机尚无法在国内使用，为此，中国移动提供了TD-SCDMA手机的租机服务。

中国电信IPTV商用详情：高清用户带宽12M

7月3日消息，中国电信公布了其IPTV商用相关资料，目前其IPTV用户已发展近百万，采用H.264标准，成为中国电信转型战略中的一个重要步骤。上海电信推出的高清IPTV采用H.264编码，视频编码速率7Mbps，音频编码192kbps，实际为用户开通的网络带宽为12M，其中2M用于宽带上网、10M用于播放高清IPTV。

小灵通转网成电信首要任务 双模不可行

电信运营商重组之后，有一个用户群需要特别关注，那就是小灵通用户。中国电信与新联通的竞争分两个方面，一是固网业务方面，一是移动业务方面。如何吸引小灵通用户转至CDMA网络是中国电信目前竞争中最重要的一环，是中国电信的首要任务。

曙光电讯IP电话超市(话吧)应用解决方案

传统的IP超市使用的是电路交换的技术，需架设E1专线，线路资源利用率较低，对带宽的需求也较大。目前，由于受到最后一公里有线接入资源的限制，大部份电信运营商只能从机房或附近的基站架设专线到业务点，而这一段线路铺设有线线路往往困难较大，每个E1线路最多能同时传输30路话音，如果话务量较大，则需多个E1线路，费用较高。因此，传统的IP超市投资成本较高，建设周期较长。

关键字 : 电信运营商IP电话局域网互连解决方案曙光电讯

三网融合背景下电信运营战略思考 （1）

三网融合背景下，运营商的网络不再是该行业唯一的物理性网络，移动互联网业务的多样性和复杂性也使得价值链演变为价值网，网络效应得以在关系型网络中实现。电信运营商失去了该效应的独占地位，营造更佳用户体验的能力将成为整个行业的稀缺资源。
