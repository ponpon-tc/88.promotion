人性定理也是人性耻辱

规范化管理有3个依据： 即人性人心定理、管理成事定理和系统思考定理，其中人性定理是 人性人心定理的简称，是指一个主体人的自我肯定原理。这一定理可以简单地概括为一句话：“任何一个健康的人的任何一个行为，都是服务于他自己的目的的”。也就是说“人的本性都是自私的、都是不可接受改变的”。所以，人性定理又有三个推论，即第一推论：人在可以懒的时候，不会不懒；第二推论：人在勤劳无益时，不会不懒；第三推论：可能发生而又不应该发生的事，却一定会发生！这三个推论在日常管理实践中随时随地、无处不在地发生。

关键字 : 人性化管理

5招教你降低辞职成本
