调查：一半的新员工对自己定价过低

CareerBuilder网站进行了一项新的调查，访问了2076位找招聘经理和人力资源专业人士，以及2999名美国工人，结果发现其中49%的员工并没有进行合同薪资的谈判，尽管有45%的雇主愿意对薪资做出一点点的调整。

关键字 : 员工雇主工资

你是“望子成龙”的经理吗？

工作与生活的平衡并不意味着你需要照顾员工下班时间的计划，而是意味着你成功地建立了一种氛围，让人感觉，如果周末没有每隔三分钟看一次收件箱，他不必感到不舒服。

关键字 : 管理员工

7步让员工敬业

站在你员工的角度来看，你会对为你工作充满激情吗？如果不会的话，也许你可以试试本文中的7个步骤。

关键字 : 绩效员工

流感季节你有预案吗？

流感季节即将来临，今年的流感强度将报复性增强。每一家公司都会受到疾病影响，但有办法让它不那么可怕，至少不必过多降低生产力。

关键字 : 生产力员工

加州保护员工社交媒体

互联网的永恒本性是，你需要谨慎你所发表的言论，无论你所在的州有什么法规。放聪明点，不要搜索员工的污点，但更不要把任何污点放在网上让你的雇主发现。

关键字 : 员工雇主社交媒体

发现欺诈不能帮你避开警觉的老板

你应该继续努力工作，而且要更加专业，同时在外面寻找工作机会。当你质疑欺诈问题时要讲实话并努力控制自己的情绪。

关键字 : 员工老板

我的工作做完才能休假？

你是豁免员工还是非豁免员工？如果是前者，那么无论你工作多少小时，你都没有义务加班。如果是后者，那意味着如果你工作40个小时或更少，你都将得到同样的薪水，但如果你工作更多，你应该得到加班费。

关键字 : 员工休假

4件事让员工保持忠诚度

如果Google和Amazon不能让自己的员工不离开，那么也证明了钱和福利不是让员工保持忠诚的答案。钱不能买来员工的爱。因此对于企业的招聘来说，最最重要的留住优秀的员工，不是付给他们高于市场基准的费用就能保证成功。

关键字 : 员工离职沟通

我的老板能合法地给我减薪么？

尽管我们很想认为其他人对于我们的困难除了同情没有别的想法，但现实情况是，当你休假时，别人接手了你的工作任务。你的缺席对你的同事和你的老板有什么影响？
