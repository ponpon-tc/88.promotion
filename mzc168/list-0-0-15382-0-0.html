香港81%住户可选择2家以上固网运营商服务

eNet硅谷动力消息香港电讯管理局日前表示，自1995年开放本地固网市场时引入互连政策，从04年7月，已开始分阶段撤销的强制性第二类互连政策，其目标已经完全达到。香港电讯管理局发言人表示，在全港共250万个住户当中，有203万个或81%住户，已可连接至最少两个客户接入网络，其中146万个或58%住户更可连接至最少三个客户接入网络。
