职场政治

兼职影响到本职工作是否告诉老板？

为了增加家里的收入，我找了一份兼职的工作，也都是用自己的下班时间来做，但是最近兼职工作需要占用我上班时间中的一点时间，我在犹豫，这件事情要不要告诉我的老板，如果我不把这件事告诉老板，好像显得不够诚实。尤其是我们还是很好的朋友。请问专家，我是不是有义务告诉他这件事呢？

关键字 : 职业道德

为工作而工作VS梦想的工作

从去年毕业到现在，我换了两份工作，每一份都是为了工作而找的工作，并不是我内心想从事的工作，我一直想从事的是电视媒体的工作，可是我知道我自身的条件和技能让我不能胜任这份工作，所以一直不敢去寻找这样的工作，站在人生的十字路口我迷茫了，我不知道该怎样去选择我的道路，希望专家给我一些建议。

关键字 : 态度

自我约束差的人如何谋划未来

我在一家公司工作，公司的环境不错，老板和同事也相处的可以。但是唯一让我感觉不好的就是薪水问题。我的工作是项目制的，项目来了就比较忙，不忙时就比较闲。我虽然想干事，但是我不能约束自己，一旦没事我就禁不住在电脑上玩游戏。常常对未来迷茫和担心，因为要养家。我如何选择未来方向，并在不忙的时候积累相关知识，以备未来呢？

关键字 : 自我管理

遭遇年龄歧视，该如何面对？

我是一个90后上班族，目前在一家国企从事汽车配件的工作，对于工作我一直抱着学习态度向前辈学习，通过自己不断的努力和学习，在工作上有很大的提高的同时也得到了领导的肯定。最近公司要评选优秀员工，我们部门领导向上级提交的名额是我，可是却被领导给否定了，否定的理由并不是我的工作做得不够优秀，而是我的年龄比较小，如果要是把这个奖项给我怕公司的老员工有想法，所以就把我给否定了，在职场上，难道因为年龄小就要受到不公平的待遇吗？

关键字 : 职场

下属遭到领导误会，该如何处理?

我在企业中是一名资深员工，专业能力、敬业能力应该都是不错的。也许因为我的一些方面相对我的老板而言，有一些优势。这也给我的老板带来了一些压力。老板在很多指示上总觉得我是和他作对，而事实上，我只是表达真实的想法和方案，也是在履行他的旨意。面对这样的情况，我该如何处理呢？

关键字 : 领导

如何在选择面窄的行业中转型

我在一家通讯领域的软件厂商负责客户需求分析、整理工作，至今已经有7、8年了。这家企业在业界还是比较不错的。原来做电信运营商的项目，可以不在现场，我虽然常出差，但时间都不长。但是最近客户的需求变了，要求我们必须在现在工作，这样一来，我几乎快变成常驻外地了，在北京的时间倒像出差了。这也极大地影响了我的个人生活，我有点想换工作。不过我所在的这个领域选择面很少，跳出这个行业，我又不知道如何选择？请专家为我支招。

关键字 : 转型

如何和老板谈判，实现内部转岗

我被现在的老板雇佣，接受了他目前安排给我的工作。但是我想转到其他部门，之所以接受现在这个工作，只是想把它当成转到其他部门的跳板。不过我不想显得忘恩负义，尤其是我真的很喜欢我现在的老板，他帮我争取到了更好的福利待遇，请问专家，我究竟应该等多久才能向老板吐露隐情，说出我非常想到另一个部门工作这件事？

如何避免接受外派而免遭遗忘

我是一名工程师，领导计划派我去海外工作，希望我去那里开展新业务。我不知道是否该接受外派，如果接受的话，外派海外一年或两年，肯定会非常有挑战性，但我的前任同事们曾经接受过类似的委派，最后他们都后悔了，说如果我接受了，会被总部 "遗忘"，最终导致我在职场上失败。请问专家对此事有何看法？

关键字 : 职场

害怕新上司质疑忠诚度，如何处之

我是一家咨询公司的客户总监，在之前有一家公司的老板想挖我过去，我一直在犹豫不决过去还是不过去。就在这中间我们公司发生了人事变动，原本想挖我过去的那个老板来到我们公司做副总，而且正是我的直接上司。他的到来让我担心他会认为我有不安分的心，以致未来影响我的职业发展。所以现在自己左右为难到底要不要离开，希望专家给我一些建议。

突破职业瓶颈的解决方案

专家您好，最近有些问题一直在困扰着我，让我不知道该怎样去做决定，希望能得到专家的一些建议，我毕业已将四年了，四年以来一直在同一个单位工作从事着销售助理的工作，可以说当初顺利的就业和很好的待遇让我有一点骄傲，但是四年过去了，我发现我的职业发展遇到了瓶颈，对自己的未来看不到发展，我很想换一份工作来改变这种现状，但是我心里一直在担心离开以后发展会越来越不好，心里很难下决定离开。
