为何Apple可能正在失去其优势？

苹果公司的问题有很多原因在于iPhone已经成为公司的利润肥缺。公司没有生产线可以接近iPhone级别的利润。如果这种模式继续下去，那么，即使苹果公司的产品销售保持强劲，公司也无法维持利润水平，更不能让投资者满意了。

成为一名价值型投资者的一种新方法？

获利型投资策略和价值型投资策略的月平均回报率分别是0.31%和0.41%，标准偏差为2.94%和3.27%。然而，同时使用这两种投资策略的投资者将会在没有额外的风险（2.89%）的情况下几乎会获得两种投资策略的收益率（每个月0.71%）。

Apple收益不足：该公司是否已经达到了极限？

苹果公司的第四季度收入刚好超过了华尔街（Wall Street）的预期，但其利润未能达到预期。从对单位产品销售额的分析可以看出，消费性电子产品（consumer electronics）仍然是主宰，虽然其销售依然强劲，但可能会成为竞争和消费者枯竭的牺牲品。

Facebook的收益：仍然存在着的两个巨大的障碍

当脸谱社交网站（Facebook）于10月23日公布了其第三季度的业绩报告，市场反应出了极大的热情。更重要的是，该公司的收益中的14%来自移动领域的消息似乎反驳了有关智能手机和平板电脑的应用程序可能会对脸谱社交网站的广告销售产生负面影响的担忧。

为什么采用“利差交易”似乎是可以盈利的
