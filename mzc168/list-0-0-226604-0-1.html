融资指为支付超过现金的购货款而采取的货币交易手段，或为取得资产而集资所采取的货币手段。融资通常是指货币资金的持有者和需求者之间，直接或间接地进行资金融通的活动。广义的融资是指资金在持有者之间流动以余补缺的一种经济行为这是资金双向互动的过程包括资金的融人（资金的来源）和融出（资金的运用）。狭义的融资只指资金的融入。融资额指融资的金额。...
