锦富新材、银河磁体今起招股 将成创业板一员

经中国证监会核准，苏州锦富新材料股份有限公司今日开启招股进程，即将成为创业板一员。本次募集资金主要投向硬盘用磁体扩建项目和高性能汽车用磁体扩建项目，预计分别投入1.38亿元和0.48亿元，合计投资1.86亿元，预计年投资利润率分别为25.05%和35.12%。

公募基金与私募基金均需客观评价

公募基金业绩相对私募基金牛强熊弱，并非一直弱于私募基金，这是由于公募基金与私募基金投资目标不同造成的。私募基金良莠不齐，业绩分化较公募基金大，需要投资者仔细鉴别，这是由于两类基金规模和投资比例限制不同造成的，公募和私募基金业绩不能简单比较。

关键字 : 基金

商务部副部长王超：并购将成对外投资重要形式
