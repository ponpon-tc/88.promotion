Familia Cassone酒庄

Familia Cassone位于阿根廷的门多萨产区。酒厂按照门多萨传统风格介绍。Familia Cassone的葡萄园有三块，分布在门多萨的西南面，其中两块位于Lujan de Cuyo，每片14公顷。在Agrelo地区有一片45公顷15年植株的葡萄园。

关键字 : CassoneFamiliaAgrelo门多萨酒庄

贝塔尼酒庄Bertani Wine

1857年，贝塔尼酒庄由两兄弟Giovan Bertani和Gaetano Bertani在意大利维罗纳北部的瓦潘提纳创立，Gaetano Bertani本身就是一个葡萄酒供应商，曾经在法国学习最新技术。在这片土地上，贝塔尼花费了50年的时间进行试验研究，旨在生产出经典华贵的“永生之酒”，最终功夫不负有心人，心血得以开花结果。

关键字 : 贝塔尼Bertani酒庄ValpolicellaWine

芭芭拉酒庄Cantine Barbera

芭芭拉家中三代种植葡萄。从那以后，酒庄致力于不断发展具有高度挑战性的研究：达到最佳质量一直是芭芭拉酒庄的主要目标，它指导着康丁.芭芭拉从葡萄园到酒厂的每一个选择。每件东西都绝对是白色的，用来聚集和放大室外彩虹的所有色彩，从而很快地传递出西西里华丽风景的感觉。

关键字 : 酒庄葡萄园BeliceBarberaCantine

法国-历时最悠久的香槟酒庄Ruinart

Ruinart是世界上历史最悠久的香槟生产商，已经成为世界各地豪门宴会上首选的香槟酒。当时慧纳香槟只是一家小厂，年产量不到70万瓶，产品只有Brut NV、Ros NV和好年份才做的年份香槟。对Barot先生而言，他在慧纳香槟的努力成果如何，这来得正是时候的奖项是最好的答案吧。

关键字 : Ruinart年份香槟Mumm酒庄Barot

意大利-贝塔尼酒庄Bertani Wine

贝塔尼酒庄的诞生与意大利的统一在同一时期，带着坚定的决心和无上荣耀，它向世人展现了一个拥有千百年古老文化底蕴的新国家杰出的成就。在这片土地上，贝塔尼花费了50年的时间进行试验研究，旨在生产出经典华贵的“永生之酒”，最终功夫不负有心人，心血得以开花结果。

唤醒建筑的潜能：让酒庄活起来

我前不久就在勃艮第(Bourgogne)发现，有一家名气很大的酒庄，把面向来访客人的直销，当作一个主要的市场来经营。新老板发现了一个比出口更容易生存、也是他更喜欢的方式：吸引客人上门来访，向他们直销。

古堡拉菲的神秘滋味

这瓶酒号称最贵的标准瓶装葡萄酒，不是“别人”，正是波尔多五大名庄之一的拉菲酒庄1787年的佳酿。拉菲古堡葡萄园酒庄拥有178公顷土地，其中葡萄园107公顷，是五大名庄中面积最大的。

美国悠久历史Erath酒庄
