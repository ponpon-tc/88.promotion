联通C网6月份增加2万户 G网增加107万户

7月21日消息，根据中国联通上市公司7月20日发布的2008年6月份统计期内业务发展数据表，截止今年6月份，联通G网用户增长到12759.9万户，联通C网则增长到4316.9万户；而6月份单月，联通增加了107万户左右，联通新增用户2.4万户。

华为确定竞标中电信C网招标：从国际杀回国内

7月17日消息，据知情人士透露，业界关注多时的中国电信C网大招标于近日启动，参与竞标的厂商众多，其中华为在CDMA领域的举动引人关注。据悉，华为参与竞标的主力基站是前段时间登顶珠峰以及德国等市场的第四代基站，让华为在下一阶段与其它厂商的竞争中信心十足。

关键字 : 华为C网厂商蛋糕基站

王晓初密会联通高层敲定C网资产划拨

7月15日消息，知情人士透露，中国电信总经理王晓初近日再度会晤了中国联通最高层商讨C网资产划拨，而结果是：联通运营C网终端的子公司联通华盛悬而未决的局面解决，将绝大部分划拨给中国电信。

关键字 : 联通C网中国电信资产属性

电信成立最大终端分销商 C网用户明年逼近1亿

尽管所收购C网资产尚未彻底清算，中国资历最长的运营商中国电信已经迅速确立最新的组织架构。据了解，该移动业务体系基本由杨小伟副总裁带领原联通系人物运作，目前除终端公司外，其他业务部门领导层已基本确定。

关键字 : 终端最大分销商中国电信移动通信C网

网通员工全面换成联通G网号码 营业厅互办业务

7月2日消息，经过一个月的过渡期，中国网通员工已基本全部更换成联通G网号码，双方也在不少地方实现了营业厅互办业务。双方还鼓励员工利用渠道、员工亲友、合作公司、大客户等资源条件大力使用和发展双方的各项业务，并按规定给予相对应的佣金。

关键字 : 网通G网员工营业厅C网

联通240人将调中电信 7月底完成C网资产确认

6月23日消息，知情人士独家透露，中国联通总部将调往中国电信的人员总数已大致确定规模，将为240人；同时，双方约定，将于7月底完成C网资产确认。根据中国联通香港红筹公司最新发布的数据，截至到2008年5月31日，中国联通C网今年新增用户122.6万户，累计用户数为4315万户；G网新增用户596.1万户，累计总用户12652.5万户。

关键字 : 联通C网中国电信调中资产

中电信为牌照让步：联通C网高低端用户流失严重

1100亿(人民币，下同)，这是中国电信与中国联通的C网最终成交价。6月2日，中国电信与中国联通宣布订立框架协议，电信上市公司将以438亿元的价格收购CDMA业务，而电信集团以662亿元的价格收购CDMA网络资产。
