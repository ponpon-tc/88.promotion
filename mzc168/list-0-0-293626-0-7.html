老板

我以前手下现在是我的老板

原手下成为老板？这会比较尴尬。你有你的价值。你很擅长做你所从事的工作。公司需要你做你现在的工作，让你一直做这个比找人替换你要更容易。

关键字 : 老板管理

我应该努力谋求加薪还是去做一些副业呢？

如果副业一个月只能带来一两百块的收入，或者如果那相当于白日梦最终你所付出的要比得到的更多，而且获得重大转机的机会并不大，这样的话你最好还是把精力都放在你的本职工作上。

关键字 : 老板加薪

老板非法克扣我工资

劳资问题永远是充满矛盾。当你超额工作时，老板不太愿意支付你更多的报酬，但是注意了，如果你减少了工作时间，你的工资单上会立刻显示出来。

关键字 : 工资老板

新来的老板是白痴

当新老板啥也不懂时，你应该更友好，更支持她。为你的个人生活设定界限，然后等待，看看会发生什么。看她是否改善，看看总经理试图解决问题。如果依然固我，你可以选择另谋高就。

关键字 : 解雇老板

我休假时老板可以给我发短信吗？

这样的事情只能美国人这么大惊小怪。在中国，这是正常不过的事情了。不过看看HR女魔头的支招，可以帮助我们既了解美国的情况，也知道该如何应对。

关键字 : 休假老板

产假回来的第一周要做到事情

于很多新妈妈来说，产假结束会百感交集。回来工作的第一周是很难回归到工作正轨上的，即使你已经习惯把宝宝留给看护来照顾了。以下是职场专家给出的能快速集中在工作上的小窍门。

关键字 : 休假老板

在断了后路以后....

把你找工作的重点放在其他地方吧。你在其他地方肯定会比在一个你最初并不喜欢的并且做出过破釜沉舟行动的公司更成功的。

关键字 : 老板人力资源

离开施虐者时你要对老板说什么？

如果你和你的伴侣结束了一段受虐的关系，你担心会对你报复，不要让尴尬的处境妨碍你寻求帮助，大多数人都愿意帮助你，而且很可能你的公司也能做一些事情，让你更加安全。

关键字 : 人力资源老板

我因为健康原因遭到解雇

如果你因为健康问题，遭到老板的解雇，下一次求职时应该如何办呢？第一步是联系你之前离开的那家公司的老板。即使你的旧职位已经有人了，也不要害怕寻求建议，以及在找新工作过程中的指导。

关键字 : 雇主简历老板

你应该在工作中讲脏话么？

根据CareerBuilder.com网站的调查，虽然有一半的员工承认他们会在工作中讲脏话，但有超过80%的老板表示讲脏话的员工看起来不专业。也许，更重要的是，有一半的受访雇主表示他们不太可能会给讲脏话的员工升职。

关键字 : 团队老板
