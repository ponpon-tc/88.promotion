我国炼油能力已居世界第二

不仅如此，中石化披露的数据显示，我国已形成17个千万吨级炼油基地，1000万吨/年以上的炼油厂原油一次加工能力已接近全国总加工能力的一半。另据国家石化振兴规划，未来新出炉的中国十大炼油基地将是：规模超过年3000万吨的大型炼油基地——宁波、上海、南京、大连，规模超过2000万吨/年的大型炼油基地——茂名、广州、惠州、泉州、天津、曹妃甸。

中石化有望提前全面供应国Ⅲ标准车用汽油

上海证券报8月27日讯记者昨日从中石化集团获悉，按照国家质检总局和国家标准化管理委员会的要求，明年起我国将全部使用国Ⅲ标准车用汽油。据中石化炼油事业部有关人士介绍，预计在三季度，高桥石化、上海石化、广州石化、金陵石化、武汉石化等炼油企业将率先完成国Ⅲ标准汽油升级工作。

中石化与台湾中油签署澳一勘探区块权益转让协议

中石化集团昨天宣布，旗下国际石油勘探开发有限公司（SIPC）与台湾中油海外石油及投资公司（OPIC）签署澳大利亚海上NT/P76勘探区块权益转让协议。2008年SIPC通过投标获得澳大利亚NT/P76勘探区块权益后，向台湾中油提出合作意向并得到积极响应。

中石化八屋-长春输气管道获准兴建

关键字 : 中石化输气管道

中石化与四大国有航运集团签署合作协议

中石化集团近日与 中国远洋、中国海运、招商局、中外运长航集团等四大国有航运集团签署长期合作协议。近年来，我国对进口原油的依赖程度不断提高，原油进口量逐年增长，保证进口原油海上运输安全具有长远的战略意义。

中石油中石化:严格执行国家调价政策

新华网北京8月5日电中国石油天然气集团公司和中国石油化工集团公司5日表示,两公司并未就修改现行的成品油价格形成机制与国家发改委相关部门进行沟通。中石油表示,7月29日国家下调汽柴油价格后,中石油严格执行国家调价要求,全力保障市场供应,在保证安全生产前提下,保质保量满足市场需求。

中石化上半年成品油销量同比下降8.43%

亚洲最大的炼油企业 中国石化昨天在官方网站披露，上半年其国内成品油销售总计5770万吨，同比下降8.43%。申银万国最新研究报告指出，预计二季度中国石化可实现每股收益0.20元，环比增长53.8%。

关键字 : 中石化

中石化沙特合资项目获批 建设成本上升20%

网易财经7月12日讯沙特阿拉伯国有控股公司SABIC于本周六表示，其与中国石油化工集团（简称中石化）设在中国天津的合作项目已经正式得到中国国家发展和改革委员会的批准。SABIC公司表示，此次与中石化的合作项目预计总成本将达到30亿美元，较6月份所公布的数字上升了20%。
