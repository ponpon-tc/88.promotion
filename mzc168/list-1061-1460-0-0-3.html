如何合理的安排工作？

我是一个部门的主管。我们部门有这样一种业务特征，要不很闲，要不很忙。闲的时候，我甚至天天打游戏，而忙的时候，又是连轴转，让人感觉压力大，很多来的人因为这个都离开了。事实上，我还年轻，也并不满意这样的工作模式，也想有更多的学习和提升。请问专家，遇到这样的情况，我该如何安排我的工作？

关键字 : 时间管理

如何管理有背景&工作懈怠的员工？

我是一个部门的中层领导，最近部门来了一位有背景的员工，至于到底什么背景，我并不知道。在实际的工作安排中，这位员工对于工作的态度极其懈怠，不能好好完成工作。而这又极大地影响了部门员工的工作积极性。请问专家，这样的员工我该如何进行管理？

关键字 : 管理员工

如何和HR沟通招聘到匹配的员工?

专家您好，公司的人力资源在给我们部门招聘的时候，并没有招聘到跟我们部门匹配的员工，往往都要从最基本的工作进行培训，不能直接上手工作，不知道是现在的人员不好找，还是人力对我们部门所需要的人才不了解，该如何去跟人力资源沟通才能更好地招聘到匹配的员工？

关键字 : HR招聘

设计稿被同事剽窃，该怎么办？

我在一家游戏公司做任务设计工作，最近发生了一件让我很气愤的事情，领导在会议上表扬了同事对新开发游戏的人物设计并作为案例展示给我们看，当看到设计时我愣住了，因为这款设计是我的作品，之前自己把手稿给同事看过，没想到他居然剽窃我的设计，我很纠结，面对自己的作品被同事剽窃，不知道该如何解决？

关键字 : 态度职业道德

工作上升空间受阻，该怎么办？

我在一家公司从事人事工作，也需要兼其他方面的工作，比如项目申报，专利等。公司是生产型企业，我在的厂是个老厂，新厂离得也不远。我需要兼这边的人事和两边的项目申报。我所在的这边基本不生产，而且这边的副总基本没什么权利。其他方面都由那边控制。现在的问题是，我一个人在这边，但是我感觉学不到东西。做的好不与不好没有人知道，做的不好也没有人批评指正。而且一个人一个部门真的很孤独。但是现在的问题是，我喜欢这家公司，虽然我有时候也抱怨它这个那个的，但是这家公司是我毕业后的第一家公司。第二是公司的领导对我很好，而且我被信任。两个公司的公章等重大机密都在我这里。（当然这有可能是那边人事斗争的结果）。第三公司调薪什么的也没有忘记过我，我进公司一年调了3次薪酬。我真的很想自己多学学，但是我在这个岗位已经停了很久。而我的人事主管第一在新厂那边，第二并没有给予我足够的发挥空间。在项目申报上，技术人员也不能够给予我足够的支持，而我本人非专业出身，就算是去学也不能做到很精通。我找了我的上司（涉及到能管我的）谈了，但是他们都对调我不置可否。我应该怎么做呢，都有点萌生退意了。

关键字 : 职业发展

如何消除和家族企业领导的不信任？

专家您好，我在一家家族企业从事职业经理人的工作，在工作的时候家族企业的领导人根本就不相信职业经理人的能力，而我一直也认为家族企业的领导在管理上存在缺陷，彼此的不信任导致不能很好地展开工作，我不知道如何消除彼此的不信任，更好的开展工作？

关键字 : 领导

职业升迁遇到天花板，该怎么办？

我是一家国企的部门经理，担任部门经理已经5年了，为了不断提高自己的能力，我参加了MBA学习班，毕业后，我发现随着学习的深入和工作经验的积累，自己却越来越觉得迷茫和困惑，感觉个人能力很难得到体现，职业升迁存在着玻璃天花板，似乎看不到有大转机的希望，面对这样的情况，我该怎么办？

关键字 : 职业发展

如何化解老板与职业经理人之间的矛盾？

我是一家民营企业的人力资源总监，我们的职业经理人很有能力，通过他部署的企业战略规划使公司发生了巨大的变化。但是，最近我们的老板和他之间出现了一些工作上的矛盾，老板似乎不那么信任他，甚至不认同他的一些做法，爆发了一场权力争夺战。对于我这个人力资源总监来说，很不想看到他们两败俱伤最终影响到公司和员工，我想问专家，如果我想帮助两位有一个良性的沟通，需要怎么做？

关键字 : 老板职业经理人沟通

性格内向要如何与人相处？

我在一家教育网站做设计工作，周末的时候我会参加一些线下沙龙，想通过这样的活动提高我在与人沟通方面的技能，但是在活动的过程中我始终没有勇气去跟其他人沟通，面对我胆小内向的性格，不知道该怎么做才好？请专家帮帮我。

关键字 : 人际关系

新的工作领域产生压力，是否要辞职？

专家您好，年后我顺利的跳槽到一家新公司做策划，因为之前策划工作我没有接触过，面对这份工作让我产生了很大的压力，我总是给自己找各种各样的理由去逃避，并且让我产生了辞职的念头，我不知道该怎么办？要不要选择辞职？
