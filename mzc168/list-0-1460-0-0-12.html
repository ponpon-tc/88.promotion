职场政治

职业经理人，空降还是内部培养？

我是一家公司的创始人，当公司资产超过20亿后，我从跨国公司挖来一些所谓的“职业经理人”。但他们大多都干得不行。我认为他们有名无实，甚至无法站在全局和战略层面思考问题，缺乏企业家精神，只是“为工作而工作”。从去年开始，我开始注意从内部培养经理人，但这个过程相对较长，如果不空降，公司的增长势必受到影响。我该怎么办？

关键字 : 职业经理人

老板拍脑袋定目标，如何巧妙应对

我在一个高新技术企业工作，负责一个基础性的部门。这个部门的工作可以说是重要但不紧急的。因为企业在高速成长，现在我这个部门的工作已经成为其他很多项目进展的瓶颈，但是在我来之前，这个部门的工作基本是空白。而我的老板认为我的工作是最简单的，希望他上午指出，下午我就完成。但这对我几乎是不可能的，因为我需要一些跨部门的合作，而部门之间还存在一些竞争利益关系。面对这样的领导，我该如何进行沟通呢？

关键字 : 领导

如何面对工作日久生厌？

我在一家企业做一个部门领导。可以说领导也比较重视我，工作氛围也非常不错。而且我们做的领域属于新兴领域。工作内容因多属项目形式，所以不是经常性的忙碌，有急有缓。薪水就我的工作而言，还是可以接受的。但是最近，我发现自己起初对工作的热情在消退了。太太天天唠叨希望我换工作来提升薪水，好在北京买房。我开始动摇是否需要换一个更高薪水的工作？我担心我要做的领域，一旦我做了不久，又失去兴趣了。请问，我该如何选择？

老员工遭遇无礼领导,如何处之?

我在一家非常有影响力的国企工作了近10年，可以说工作上是比较认真的，而且我对当领导没有太多想法，只想工作环境比较融洽，而且有我一份实在的工作做，但是最近我这个部门让我非常心烦。我的领导是那种经常当众侮辱我的人，说我“这点小事都不会干。”不仅对我，部门同事也都如此。我是老员工，也有自尊心，对这样的领导，我真不知道如何和他相处。而现在外部的就业环境又不是太好。请问专家，我该怎么办？

关键字 : 领导者素质领导

入错行后的转行之痛

我本科读得是物流专业，也考取了物流师的各种证书。我毕业那年正是金融危机如火如荼的岁月，找份工作实属不易。尽管我热爱本来专业，可当我收到现在供职这家外企marketing的offer时，我还是毫不犹豫的选择了现在的工作机会。工作除了经常出差和工资不特别高外，其他的还是很理想的。但随着干得时间越来越长，我就有一种对这份工作说不出的反感，这根本不是我想要的！我对工作渐渐地不再有目标，仅仅是麻木的做事，把这份工作当成赚钱和谋生的工具。同学同事都劝我趁最近就业机会好，赶快跳槽。但我舍不得积累起的人脉，而且干回物流就意味着一切从头开始，我这些年的经验经历毫无用处。我很纠结也很彷徨，想请专家帮忙支支招。

不想被当枪使，又想左右逢源？

最近我们企业新董事长上任，上任第一天就把我们这些中层都叫去开会，对我们的批评远多于表扬。而且最为让我费解的是，董事长先让我做一个5年的部门业务规划，然后才让总经理做一个3年的企业规划。这让我很为难。因为管理的人都知道部门业务规划应该在企业规划之后制定，我不想夹在董事长和总经理之间，而且我也怕被当枪使？请问专家，我该如何应对呢？

关键字 : 领导

无实力创业者如何用商业谈判掘金

我是个在上海工作了3年的本科生，在一家中型服装对外贸易公司从事网站编辑和美工工作。现在准备自主创业，做服装或食品方面的中间商。通过3年的工作我积攒了一些这方面的货源和人脉，可毕竟没有当过老板，对于该怎么和零售商谈判这件事非常头疼。自己的创业路刚起步，公司和自身都不具备很强的实力，说话也没底气。如何通过谈判打开市场是我最最关注的问题。所以我想请问专家，面对创业初期的商业谈判，我该注意哪些方面呢？

关键字 : 创业

新单位实习:选择高调还是低调?

我有个烦恼是，我不知道我在实习的时候是应该特别积极，还是有所保留。因为积极的话可以显得态度好，但是也容易显得锋芒毕露；可也不能显得太低调，这样显不出自己的价值。我想知道这个度该怎么把握，请专家帮我支招。

升职承诺几年不兑现,走人?

我在一家企业的中层岗位已经干了6-7年了。我的业绩、成绩是有目共睹的，能力也得到领导的认可，并且让我管理的事情似乎也越来越多。不过我有点郁闷，就是最初老板让我干时，曾经许诺不久后，让我当公司的副总，但是已经几年过去了，似乎这事没了消息。我现在30多岁，我不知道这样还要熬多久？我想离开。不知道专家有什么建议可以给我？

如何不成为公司派系斗争的牺牲品

我刚到一个新单位不久，但是在这些日子和同事的沟通中，我已经发现这个单位存在着不同的派系。已经有不同派系的人向我示好，希望我能站在他们的队伍中。而我是最讨厌这种的拉帮结派，我认为公私最好分开。这样的工作环境让我感觉非常困惑。请问专家，我该如何面对？
