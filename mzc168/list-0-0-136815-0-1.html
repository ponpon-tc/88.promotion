你属不属于加薪人群？

为此，笔者采访了萨蒂扬软件技术(上海)有限公司人力资源经理田静和南美轮船(中国)有限公司的中国区人力资源总监唐蓓莉，让他们从HR的角度揭开加“薪”的奥秘。一般来说，加薪分为几种形式，一是企业根据整个行业薪酬状况对员工薪资进行调整；二是企业根据自己的薪酬体系对部分员工进行加薪。

年末加薪大战

近期，人们最怕听到的就是：菜价涨了、肉价涨了、油价涨了、日用品涨了……去年，CPI连创新高，每个职场人都揪着心，期待着2011年工资的上涨幅度能赶超物价的上涨幅度。对企业而言，在有限的预算内，会根据各种指标把员工分成三六九等，分别给予不同加薪幅度。

今年上海企业平均加薪将达8.8%

在新的一年到来之际，职场人都在企盼企业能够加薪。不同发展阶段的企业加薪幅度也不相同，处于快速发展期的企业加薪幅度相对较高，部分企业的加薪幅度甚至超过15%，而处于稳定发展期的企业加薪幅度较平稳，一般在6%-10%之间。
