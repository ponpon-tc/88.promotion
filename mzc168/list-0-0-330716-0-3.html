互联网之后，会是物联网吗？

计算模式每隔15年发生一次变革，这个被称为“十五年周期定律”的观点，一经IBM前首席执行官郭士纳提出，便被认为如同摩尔定律一样准确。的确，纵观历史，1965年前后发生的变革以大型机为标志，1980年前后以个人计算机的普及为标志，而1995年前后则发生了互联网革命。
