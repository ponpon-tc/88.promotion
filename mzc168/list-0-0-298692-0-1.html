联通放号第一天:上网套餐自动升级

中国联通3G无线上网卡具有套餐自动升级功能，将带给您更贴心，更实惠的通讯服务。套餐自动升级是指：系统每月先扣除客户所申请套餐内包含的数据流量，超出部分按照流量计费；当套餐月使用费与超出部分使用费之和达到更高一级套餐月使用费时，系统将自动向更高一级套餐升级。
