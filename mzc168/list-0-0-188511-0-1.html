雪利白兰地产区——欧八马葡萄酒业

雪利白兰地产区因其悠久的传统和独特的加工方法而成为欧盟的三大白兰地法定产区之一。上个世纪后半段，烈酒贸易的发展在经历了一段时期的停滞后，雪利白兰地贸易开始复苏，并出现了瓶装雪利白兰地。

关键字 : 白兰地产区葡萄酒业Solera橡木桶

制造酿造葡萄酒的橡木桶的材料通常采用法国的Allier和Nevers地区生长的橡树，采用这种橡木制成的橡木桶酿酒，酿造出来的葡萄酒更加醇厚。（2） 由于葡萄梗内的单宁的含量很大，习惯上在葡萄压榨之前要去除梗，因为一旦浆果破裂，发酵就开始了。

里奥哈葡萄酒分级品尝（组图）

1926年,里奥哈（Rioja）成为首批西班牙法定产区之一,1999年第一个获得DOC产区级别的认证,堪称西班牙国酒,本期精选里奥哈地区三个不同的陈酿等级的代表美酒,让您慢慢品味西班牙葡萄酒的旖旎风味。

关键字 : 里奥哈品尝产区Reserva橡木桶

里奥哈葡萄酒分级品尝

Reserva（珍藏/陈酿），干红葡萄酒要求至少陈年3年才可以面世，在橡木桶中陈年不少于1年，干白和桃红葡萄酒要求至少陈年两年，在橡木桶中陈年不少于6个月；Gran Reserva（特级珍藏/优质陈酿），干红葡萄酒要求至少陈年5年才可以面世，在橡木桶中陈年不少于18个月，干白和桃红葡萄酒要求至少陈年4年，在橡木桶中陈年不少于6个月。

关键字 : 里奥哈品尝产区Reserva橡木桶

酒标的秘密之西班牙

同法国、意大利一样，西班牙也是老牌旧世界葡萄酒国家，在2004年，西班牙更被评为“葡萄种植面积世界第一”。1972年，西班牙农业部建立了原产地名号监控制度，并对西班牙葡萄酒进行了金字塔式的分级：　　1. DOC/ DO，相当于法国的AOC级别。

关键字 : 西班牙酒标橡木桶葡萄酒Vino

西班牙葡萄酒巡礼之二

人工挑选原料，选用先进设备压榨尽可能地保留果实中的各种天然成分，控温发酵，4个月橡木桶陈酿，装瓶后再经过至少10个月窖藏才上市销售　平衡的酸度很好地结合了橡木中的丹宁成分，口感平衡，丝滑，稠密。

关键字 : 控温发酵橡木桶陈酿葡萄园栽培技术

好喝又好看的西班牙葡萄酒

品种介绍 tempranillo（添帕尼），意思是早熟葡萄，是里奥哈产区种植面积最广的品种，酿造的红葡萄酒在世界上享有声誉。酒评 酒体好，呈樱桃红，轮廓偏紫，香气浓郁，有水果，青椒，热带植物以及焦糖的气味。葡萄未经压榨直接发酵，橡木桶中陈酿4个月，酒瓶中陈放12个月以上。

关键字 : 香气西班牙产区品种介绍橡木桶
