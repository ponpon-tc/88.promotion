欧元

欧元硬币 欧元（Euro）的前身是欧洲货币单位，现在是欧洲货币联盟（EMU）国家的单一货币。欧盟27个国家当中有16个国家使用欧元，它们是奥地利、比利时、芬兰、法国、德国、希腊、爱尔兰、意大利、卢森堡、荷兰、葡萄牙、斯洛文尼亚，西班牙，马耳他，塞浦路斯和斯洛伐克，这些国家合称为欧元区（Eurozone）。在欧洲，每天有超过三亿人使用欧元。在欧洲以外，有23个国家和地区把自己的货币与欧元挂钩。欧元在香港被称为欧罗。欧元的符号...  


欧元有多脆弱？

对于欧洲的经济增长，按人均计算，欧洲经济的增长速度实际上超过了美国。欧元区成员国面临着较大的困难，但它们可能并不比美国所面对的具有更大的挑战性。

关键字 : 欧元债务

希腊的邻国是更大的问题

希腊的问题是公共债务约占GDP的160%，而且还在持续增长。因为希腊相对较小，市场真正担心的问题是"蔓延"，危机已经扩散到较大的经济体中，如西班牙和意大利。这些国家都有较高的债务水平，需要通过出售国债来再融资。

关键字 : 国债欧元投资

欧债危机,我们如何到达这一步的?

与2008年美国的金融危机不同，欧洲主权债务危机展开得非常迅速，并且这一最新的金融危机仍然还在继续着……不断……不断地继续着。

关键字 : 欧盟欧元债务

如果希腊离开欧元区,会发生什么?

如果希腊拒付其债务并退出欧元区会是怎样的状况呢？这对希腊周边的国家将会产生很大的影响，但是没有人能知道这个影响会有多大。而对于希腊本国来说，这可能是最快最好的重建国家经济的方式。

关键字 : 欧元债务

什么推动希腊退出欧元区？

虽然没有人知道如果希腊拒绝偿还债务并退出欧元区将会发生什么，但前景肯定是可怕的。本文概括说明了为什么这个经济不景气的国家可能会退出欧元区货币联盟以及这样的情况可能会怎样发生。
