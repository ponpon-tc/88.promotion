职场政治

如何争取晋升机会？

我在一家公司的销售部门已经工作三年多了，在此期间，我从一名销售助理成为一名优秀的销售人员。我很努力的工作，也有一些成绩和客户，现在部门缺一位销售经理，我很想去争取这一管理职位，也很想尝试一下，请问专家，我该怎么去争取这个职位获得晋升呢？

关键字 : 职场

如何管理出现内部矛盾的团队？

我是一个企业的销售总监。为了培养销售队伍，我将销售部门分成了两个小组，由两个比较好的销售人员来带一些年轻的销售。但是最近一个销售小组的组长因为其小组一个成员在一些方面不好管理，就鼓动小组其他成员与该员工分裂。并且一再向我提出不合理要求，比如要一些容易出单的客户，把不容易的客户踢给其他同事。当我没有答应他的时候，他就在微博上发布各种不利于团结的信息，并且威胁让公司开他。请问这样的人，我该如何管理？

关键字 : 团队管理

创业老板该如何突破管理的局限？

我在创业企业的老板，公司已经有稳定的收入和发展了。最近我们融到资金，准备扩大规模。对于我而言，我习惯于管理小的团队，因为我可以了解他们的背景、家庭、成长经历等，我感觉像家人。但是一旦公司成长到百人以上，我感觉对我的挑战很大，因为我不可能了解那么多人，请问专家，面对我这样的情况，什么方式可以帮助我突破我自己的局限？

关键字 : 创业管理

发展空间受限制&离职，该如何选择？

我是一家公司的中层管理人员，在我的岗位上做了5年，带领我的团队每年都实现30%到40%的销售额增长，所有的考核也都还不错，我一直想加薪升职，但就是得不到老板的提拔，总是找各种借口推辞我。我很喜欢我目前的工作，但人都是希望自己的付出和得到成正比，请问专家，我是应该接受目前这种没有任何发展的空间还是应该离开我所喜欢的工作岗位到其他公司去寻求晋升的通道？

关键字 : 职业规划

是否可以到同一个公司面试两次？

我是一名失业人员，正在找工作，我看到一家我向往的公司正在招聘，这家公司我两年前投递了简历，并且参加了面试，但面试两轮后就被淘汰了，如今他们又在招聘和两年前一样的职位，现在好多公司都简历存档，我担心上次应聘的失败会成为我这次应聘的劣势，请问专家，我现在还能再次投递简历参加竞聘吗？如果可以，我该如何去说明？

关键字 : 面试

如何和故意找茬的领导相处？

本人是部门经理，我和我的领导（公司副总）一起共事有8年；平时在工作和沟通上都没有问题，但是最近我发现领导总是在故意找我工作的问题，我跟他汇报过的工作进程，他说我没有汇报，并且还在同事面前挑我的错误，我又不好当面反驳，弄得我现在很被动，不知道该怎样去和领导相处。

关键字 : 领导职业道德

与老板发生冲突，选择集体离职是否正确？

　　专家你好！我在一家电子商务公司的客户服务部门工作。最近由于公司要迁往外地，希望员工也能选择，随公司同迁，或者离职。我们在和老板沟通的过程中，发现老板一直都没有给我们上应该的保险。我们也不愿意离开这个城市，于是员工和老板之间爆发了一场巨大的冲突。我们决定集体离职，我已经向公司递交了离职报告。请问专家，这种集体离职是否正确？对我有意义吗？另外，我是应该找到工作在辞职，还是辞职以后再找？毕竟我现在的工作是每周六天，基本不可能请假去面试。

关键字 : 离职

公司高层领导有变动，员工该怎么做？

你好专家！我是一家大型基金公司的中层。最近公司高层发生了巨大的人事变动，我们的一把手，以及我们公司的标志性人物相继离职。这给公司上下带来极大的震动。因为是在国企，我们不知道下一个头是谁，会对公司有怎样的影响?虽然公司内部也会有一些派系，由于我的业务比较边缘，我似乎没有在任何人那里站队。现在公司每天都乱哄哄的，每个人不知道做什么，也不知道怎么做。请问专家，我在这个时候应该怎样做是最好的？

关键字 : 员工领导

与空降领导发生业务冲突，该怎么办？

我在外企的一个部门工作了多年，最近我们部门从别的公司空降了一个头，这个头的风格和我们以往的文化和风格有很大的不同。最为关键的是，他并非常了解我们这个部门的业务情况，但他又非常喜欢表达自己的意见，并且急于要做出业绩，不太愿意听我们的建议，所以我们之间常常会有一些冲突。面对这样的状况，该如何相处？

关键字 : 空降兵

对职业发展感到迷茫，该怎么办？

您好,我有个很纠结的问题就是我不知道我适合什么样的工作??!!在这毕业的一年里换了两三份工作,我发现我总是找不到合适的岗位,也不知道自己到底适合做什么,所以总是左右摇摆.有人说这是"90"后的通病,似乎频繁跳槽成了我们的代名词,但其实最主要的问题是我们不清楚自己目前的职业发展状况吧.你说遇上这种问题应该怎么办...?
