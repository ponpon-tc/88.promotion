“AWS领先优势结束” OOW 2016的重点是IaaS

“AWS的领先优势已经结束。”在2016年Oracle OpenWorld上，甲骨文公司董事会主席和首席技术官拉里·埃里森将第二代IaaS作为重点展示给与会者，并向AWS宣战。

ZD至顶网软件频道消息： 

 “与AWS相比，甲骨文IaaS的计算能力高2倍，内存多2倍，存储大4倍，I/O多10倍以上，而成本则比AWS低20%。AWS的领先优势已经结束。” 甲骨文公司董事会主席和首席技术官拉里·埃里森（Larry Ellison）在2016年Oracle OpenWorld（OOW）大会上将AWS视为头号劲敌，并正式将Oracle第二代基础设施即服务作为通用IaaS提供给企业用户。

Oracle OpenWorld 20年 仍火力十足

美国当地时间9月18日，Oracle OpenWorld 2016在美国旧金山市中心正式拉开帷幕。对于红色巨人来说，今年的大会也是格外特别的一届。 

20年前也是在这个城市，第一届Oracle OpenWorld正式举办，在20年间，大会不仅成长为全球最大规模的技术大会，同时也累计为旧金山市创造了30亿美元的收入，即使是当地的餐馆、酒店也都会在每年的九十月间参与到大会的支持中来。

在去年的Oracle OpenWorld上，拉里·埃里森第一次全面描述了Oracle的云蓝图，即集成云应用及平台服务，将Oracle Cloud描述为全面的SaaS、PaaS、IaaS三层的服务，并将Workday、Salesforce、微软以及AWS作为此间的竞争对手。 

从过去的一年的发展看，虽然Oracle云业务的营收比重在整体营收比重中仍较低，但是其增长态势很强劲，尤其是SaaS、PaaS服务的营收占了云服务营收的大半。最新的2017财年第一季度报告显示，其云业务总收入为9.69亿美元，其中SaaS和PaaS收入为7.98亿美元。

甲骨文公司首席执行官马克·赫德（Mark Hurd）指出：“甲骨文SaaS和PaaS销售额的亮眼表现，已经连续第二年超过任何其他云服务提供商。仅在第一财季，我们就新增了750多家SaaS客户，包括344家SaaS Fusion ERP客户，比Workday自成立以来所有的ERP客户数量还要多。”

而在今年的Oracle OpenWorld中，拉里·埃里森对其云战略进行了调整，不仅火力十足地将AWS视为头号劲敌，还重点强调了在PaaS和IaaS领域与AWS的竞争。在他看来：“IaaS是一个极关键的机遇，可让其在快速增长的SaaS和PaaS业务基础上实现更好的发展。”

明知山有虎 偏向虎山行

虽然拉里·埃里森的火力十足，但是现在云的市场竞争却并不轻松。

根据Wikibon的最新统计，2016年上半年全球IaaS公有云的市场份额中，AWS占31%，第二名Microsoft占24%，然后是IBM的14%，其他厂商都没有超过10%。Oracle要想在其中跟AWS竞争，还是要狠下一番功夫的。 

据了解，最新发布的Oracle第二代基础设施即服务包括了Oracle Bare Metal Cloud Servers、Oracle Ravello Cloud Service和Oracle Container Cloud Service，其中，Oracle Bare Metal Cloud Servers要比AWS的解决方案快11.5倍，并且成本降低20%，同时支持数据库即服务（DaaS）、网络块存储、对象存储和虚拟专用网连接。

Oracle Ravello Cloud Service可以支持企业客户不进行任何修改配置的情况下在公有云中运行VMware。Oracle Container Cloud Service为企业提供了一种与Docker兼容的方式，通过一次点击即可部署应用程序堆栈。

值得一提的是，Oracle Bare Metal Cloud Servers还支持企业本地部署，与Oracle Cloud实现无缝连接。

除此之外，在拉里·埃里森的演讲中还发布了Oracle Database 12c R2。数据库作为打败AWS的秘密武器而重点提及。拉里·埃里森表示，Oracle数据库的优势就在于可以运行于各种平台，不管是Oracle Cloud、AWS，亦或是微软Azure，统统运行无压力，但是AWS的Redshift数据仓库却只能运行在AWS环境中，企业用户一旦使用就“跑不掉了”。

据了解，Oracle Database 12c R2现在可以支持Exadata Express Cloud Service，11月将支持Enterprise Cloud Service和Exadata Cloud Service，12月将支持Exadata Cloud Machine。12c R2比12.1性能高60倍。

竞争很重要，但是当下更重要的是确保企业能够顺畅地从内部部署向云转移。拉里·埃里森表示，当下是一个企业从内部部署向云转移的时代，但是这个变革却不是一蹴而就的，在未来10年中，企业客户需要各种部署模式的服务和产品支持，而最重要的是打通这些部署之间的联系，确保企业客户能够快速顺畅的从内部部署向云转移。

根据Gartner在2015年有关云采用的调查发现，企业采用公有云可以节省下14%的预算，但是，对采用云服务的愿望却赶不上实际的使用。毫无疑问，对于企业来说，迁移到云仍然是存在挑战的。

为了企业能够更顺畅的向云迁移，拉里·埃里森为Oracle Cloud设立了六个目标：成本、性能、标准化、兼容性、可靠性以及安全性：

  • 成本：最低的采购价和最低总拥有成本；
  • 可靠性：强大的容错能力，实现无单一故障点；
  • 性能：最快的数据库、中间件和数据分析；
  • 标准化：实现从SQL、Hadoop、NoSQL数据库，到Java、Ruby、Node.js、PHP等编程语言，以及Docker、Linux等的标准化；
  • 兼容性：从内部部署到云上轻松迁移，确保应用程序的可移植性和数据的便携性；
  • 安全性。安全可能是企业从内部部署的向云迁移所面临的最重要的问题，要确保始终在线的、连续的网络攻击防御。

年过七旬的拉里·埃里森用了整整70页PPT阐述Oracle云服务的蓝图，不仅是IaaS，还有SaaS和PaaS，在未来的三天内，Oracle的团队和合作伙伴一起将为大家呈现出一个在云方面更锐意进取的Oracle。
