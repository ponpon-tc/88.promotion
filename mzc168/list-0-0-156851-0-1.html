浪漫的阿尔萨斯白葡萄酒

阿尔萨斯（Alsace）是法国最顶级的白葡萄酒产区，也出产高品质的起泡酒。阿尔萨斯葡萄酒酒味纯粹干净、口感宜人，均为法国法定产区(AOC)酒，这也是法国唯一一个容许在法定AOC葡萄酒瓶上标注品种的产区。

德国：白葡萄酒的故乡

西班牙白葡萄酒巧制爽肤水
