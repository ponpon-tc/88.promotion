职业兴趣&工作压力，该如何选择？

我是一个刚毕业没多久的大学生。摆在我面前的有两条截然不同的职业发展道路。一条路是我喜欢的，但是当我真正开始做的时候，我感觉有压力和力不从心。另一条是可能压力小点，但是却对这样的工作不感兴趣。我非常迷茫，在左右的摇摆中，但我也知道我不能长期处于这样的状态中，请问专家现在的我应该怎么做？

被部门领导孤立，该如何处之？

我在一家国企上班20年了，由于和直接领导的关系没有处理好，（曾经对我领导进行越级投诉），结果我现在这个部门被这个领导孤立，并且这个领导用各样的方式想逼我离开这个部门，处处整我。我的心情非常不好。因为毕竟在这个企业已经工作20年了，如果让我放弃这里，出去开始新的工作，我心不甘。但是在这里，我又极其不舒服。请问专家，我该如何处之？

关键字 : 领导员工

公司的薪酬体系不完整，该怎么办？

我到一家公司工作一段时间了，但是我的薪水一直没有改变。因为入职的时候，也没有问这个企业的薪酬体系是怎样的。现在遇到这样的情况，我很想知道如果我想了解薪酬体系问题，我应该怎么办？如果公司没有好的薪酬体系，（据说周围很多同事都是几年没有涨工资），请问专家，这样的公司我是否值得继续工作下去？

关键字 : 薪酬体系

海归如何做好职业规划？

我是一名海归，4年前充满信心回到祖国，回国找了两次工作，但都在试用期刚结束就不再聘用我了。第三份工作好不容易站住了脚，并且积累了一些人脉和工作经验。可是随着年龄的增大，我认为自己在事业上没有显著的进步。请问专家，我该如何更上一层楼呢？

关键字 : 职业规划

跳槽以后如何快速开展工作？

我最近跳槽到之前公司的竞争对手那里上班，这里给的薪水和职位都比原来公司高，但面临一个巨大的挑战，就是我即将要带领的部门收入和盈利都在大幅下滑，属于全公司效益垫底儿的部门，新老板和新同事都期待我能够立刻展开工作并年底实现盈利，当下需要做哪些工作，不应该做什么哪些工作？在此想请教BNET的专家团队，谢谢。

关键字 : 职业发展跳槽

求助！开店和工作如何兼顾？

专家您好，我是国企的员工，安逸稳定的工作让我感觉到压力很大，我很清楚要是一直这样下去我的职业道路会是什么情况，所以在经过很长时间的考虑，自己想要开一家小吃店，因为创业的经验不是很多，我不知道是要把工作辞掉专心的去做，还是要一边工作一边开小吃店（我个人比较偏向于第二种）希望专家可以给我一些建议？

关键字 : 职场

如何缓解不被领导重视的情况？

我从事的是市场推广的工作，跟着现在的团队我能学到很多东西，但是我总是感觉到我一直再被领导忽略，像一些活动和推广的工作领导从来都不征求我的意见或是想法是什么，每次这个时候我心理就很不舒服，总感觉自己是个局外人，我不想离职，但是也不知道要怎样缓解这种情况？

关键字 : 职场领导

离职日期已到公司不让走，该怎么办？

专家您好，在半个月前我向公司提出辞职，公司批准了我的申请并且同意我把手上的项目做完以后离职，现在我手上的项目已经做完了，公司却没有按照之前的约定让我走，而是说在没有招到人之前我不可以离职，面对这样的情况，我该怎么办？是不管不顾的直接走，还是等待？

关键字 : 离职职场

同事不配合工作很被动，该怎么办？

我在一家公司的一个项目组中工作，我的组长因为工作任务很多，决定分配一些任务由我来负责。虽然这些任务名义上有一些同事来协助我完成，组长虽然没有下达明确的授权表示这些同事由我来分配任务，但是也在会议上有过说明。但是我发现任务到我这里，就分配不下去了。那些同事都认为做不好，又把任务还给我了。结果我很被动，被工作压的喘不过气。向组长说明，他认为我没有做好工作，并说如果我对谁不满意，可以告诉他，给辞退。可是开人不是我愿意做的，请问专家，我应该怎么办？我真的很纠结。

关键字 : 职场

如何面对老板的不公平待遇？

我是一家基金公司的门市店负责人。年初老板给每个网店下达了按月的销售任务，并制定了相应的销售考核指标。我根据我们网店的实际情况，申领了一定的销售任务（这个任务不是所有门店中最高的），并和老板签订了协议书。如果完不成任务，我们每月绩效将受到很大的影响。经过我们门店的努力，我们终于超额完成了任务。如果按照当初的任务指标，以及我们完成的情况，根据销售考核，我们应该拿的系数是1.8，但是最后老板却只给我们1.2。受到这样的不公平待遇，我和我的下属都感到非常气愤，请问专家，我应该如何面对这样的不公平？

关键字 : 待遇职场
