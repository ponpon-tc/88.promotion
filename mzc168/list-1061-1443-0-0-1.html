小心！年底盲目跳槽易致心理抑郁

“跳槽”在当今职场中已是司空见惯的事情，特别每年年底，是跳槽、换工作的高峰期。跳槽还使人丧失成就事业最宝贵的敬业与团队精神，容易浮躁，凡事浅尝辄止，这样跳来跳去，最终一事无成。

关键字 : 跳槽心理抑郁

职场自测：年底你成了工作狂了吗？

在这个竞争剧烈的社会，每个人几乎都快被忙碌的工作所淹没，“工作狂”早已成为这个社会的普遍现象。相反，如果能够合理安排或是将一些家务留到周末再处理，就能使做家务成为工作之余的放松手段。

关键字 : 工作狂职场

测试：你有赚钱的心理基础吗

如果你的得分在12分以上，这意味着你已经具有一定赚钱的心理基础了，可能你还具备了较强的赚钱能力，你可以考虑选择一个项目大胆地去干。如果你的得分在12分以下，那么，你在准备投身于某一个项目之前，不妨再学习或训练一下自己的赚钱技巧吧。

关键字 : 赚钱能力心理基础

趣味测试：你会是个公关精英吗(图)

一名杰出的公关精英，就得在各种场合，面对各种人，处理任何事都游刃有余。良好的公关能力同时也是现代社会生活中，交流的重要能力。一起来测试一下吧。点击图片进入测试 本测试仅供娱乐　

关键字 : 趣味测试公关能力

趣味测试：你会拥有“铁饭碗”吗(图)

所谓“铁饭碗”，顾名思义，饭碗乃铁所铸，坚硬非常，难于击破。长期以来，“铁饭碗”一直为人们所羡慕和追求，若捧得此碗，从此便可衣食无忧，生活幸福。快测测你能不能拥有铁饭碗吧。

关键字 : 趣味测试铁饭碗

事业测试：公司裁员黑名单有你吗(图)

供稿/天承我心工作室　 欢迎关注新浪星座微博　　本想不再跳槽，就在这家公司好好干下去了，可是没有想到还要在担惊受怕中熬日子，因为公司裁员的警报已拉响，据说还列出了裁员的“黑名单”，大家都面临裁员的危险，在没有正式公布之前，人人自危。

关键字 : 黑名单公司裁员

事业测试：你是否成了职场橡皮人(图)

独家撰稿：噬·碎魂 欢迎关注新浪星座微博　　蜗居的压力，被炒鱿鱼的威胁，飞速上涨的物价，无时无刻不给我们带来莫大的心理压力。

关键字 : 橡皮人职场

你是属于哪种职场性格

虽然有些言过其实，但在如今的职场上，个人的“职场性格”因素的确非常重要，它决定着你的职场表现，影响着上司对你的看法，左右着你的职业发展。了解自己的“职场性格”，并对症下药进行适当的调整，将有助于你更好地把握职场机遇。

关键字 : 职场性格

测试：怎样减压才最有效

你接受朋友的邀请，一起乘船出海去玩。选C 当你感到郁闷或是心情黯淡的时候，不妨以你为中心办一个聚餐或是唱唱卡拉OK也不错。这样过个两、三天之后，等到你的心中有“想做事的感觉”时，再去恢复你的工作。

关键字 : 测试减压

你的工作态度让人满意吗

选B：你对工作企划有一套，眼光远大，能安排好一个月以后的行程，只可惜你做事太保守，缺乏冲劲，不能专一地投入，不然你为何贪恋山谷的美景，而不把全部心神投注在钓鱼上。选C：工作狂热症的代表，就像追求坐船时乘风破浪的快感一样，你是一股劲儿地拼命，也就是说，拼命起来没大脑，你只能听指令行事，但是绝对不能让你规划，因为你会急出脑溢血。

关键字 : 战术
